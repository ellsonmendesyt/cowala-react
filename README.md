## COMO RODAR O PROJETO
- clone o respositorio 
<pre> 
git clone https://github.com/ellsonmendesyt/cowala-react.git
</pre>

- Instale as dependencias 
<pre>
 yarn install
</pre>

- Rode o projeto 
<pre>
 yarn start
</pre>
